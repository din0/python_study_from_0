# 第十章 输入输出

1. 输出

    print()
    将输出的值转为字符串：
    str(): 返回一个用户易读的表达式
    repr()：产生一个解释器易读的表达式，参数可以是任何对象

    ```python
    >>> s = 'hello world!'
    >>> s.title()
    'Hello World!'
    >>> str(s)
    'hello world!'
    >>> repr(s)
    "'hello world!'"
    >>> print(s)
    hello world!
    >>> list = ['asd','qqq','222']
    >>> repr(list)
    "['asd', 'qqq', '222']"
    >>> tuple = ('11','aa')
    >>> repr(tuple)
    "('11', 'aa')"
    >>> dict = {'a':22, 'c': 44}
    >>> repr(dict)
    "{'a': 22, 'c': 44}"
    ```

    ```python
    >>> for x in range(1,11):
    ...  print(repr(x).rjust(2), repr(x*x).rjust(3), repr(x*x*x).rjust(4))
    ```
